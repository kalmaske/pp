NYT Scraper Project

Web Scraper

The application scrapes the NYT website for new articles and adds them to a Mongo Database. There is a "scrape new articles" button for scraping. After scraping the app displays the headline, summary, and URL of the each article.

The user is ale to save articles and adds notes to them. Notes can be revised or deleted. All saved articles and comments are visible to every user.

This application was built using an express router, CSS, Bootstrap, Handlebars, and NodeJS. npm packages used: express, express-handlebars, mongoose, body-parser, cheerio, & request.
